# Build Snap (all buildings)

**Description:** Snaps building placement to the nearest valid build grid point.

Just like building metal extractors, this widget will snap the building to the closest valid building position. 
This especially makes building close to water or on cliffs much easier. 

Also easier to fit buildings closer together or be generally quicker in placing large buildings.
