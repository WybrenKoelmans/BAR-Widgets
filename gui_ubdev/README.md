# Cheat Interface

**Description:** Provides a GUI for cheat functionalities.

Do you always forget that ticks are called fleas and Scuttles are actually spelled Skuttles? Do you like to click buttons? Are you not afraid of bugs?

Then this is the widget for you!

Note: this widget is intended for developers, I provide no player support on this widget.

Instructions
To move widget around, drag "uBdev" with mouse. 
To place units using /give panel, click a unit in the panel, then click in world (can repeat for more units), right click to clear selection.

Filter
[x]: Clear all filter
Input field: search for unitDef.name or ingame name
T1, T2, T3: tech tier filters
B, U: B=buildings, U=units only
A, C, L= filter by faction
Filter can be combined

At the top you will find always find some useful units (you can tweak them by adjusting the favorites var)

[remove selected]: any units selected in the world will be removed.
