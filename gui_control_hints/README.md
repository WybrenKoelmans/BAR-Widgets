# Control Hints

**Description:** Show little hints about controls when in the proper context

This widget will show hints when commands are active like RECLAIM, REPAIR, etc. to show their modifiers and what they do. 

For example, did you know that holding down CTRL + DRAG for a RECLAIM command will prioritize metal? Yes? OK what about holding down ALT so Shuriken don't automatically change target after paralysing? I bet this widget will have a surprise for you. 

There is also API available for other Widgets so they can add their own Hints as well.
