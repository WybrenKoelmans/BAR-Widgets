# More Range Indicators

**Description:** Shows vision range indicators with shader rendering.

Ever wondered where the hell to place a camera on a cliff so you can see on top of the cliff? This widget will tell you with about ~80% accuracy! (It's a technical thing, I would have liked it to be 100% but I'm not sure it's possible to recreate the engine LOS logic in a shader).

Also good for noobs, it visualizes that some defence turrets have greater weapon range than vision, so you should build a camera/radar next to it to maximise damage output until you have pinpointers
