# Resource Panels

**Description:** Displays detailed, customizable resource usage panels for all units, including live per-unit and aggregated resource flows (metal and energy). Features world-space labels, overview panels, clustering, and color-coded drain/income indicators for advanced resource management.

Press F7 (default) to enable Resource Mode screen and you will have panels that will show you where your energy is going. 

I might still need to tweak this a lot, but it's quite functional already, so feel free to leave suggestions.
