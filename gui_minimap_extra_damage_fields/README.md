# Minimap Extra - Damage Fields

**Description:** Enhances the minimap with damage visualization that fades over time

This widget records incoming damage and visualizes it on the minimap as pulsating areas on the minimap. It stays around for a while so if you happen to lose a few units when you are not paying attention, you can see where the battle happened.
It also works for units on players on your team, so you can more easily see that they are being attacked (and potentially already lost and need help).
