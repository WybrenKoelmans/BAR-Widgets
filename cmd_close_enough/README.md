# Close Enough

**Description:** Clicking close to a unit/structure will select it anyway, close enough!

If you misclick next to a unit (or have a difficult camera angle), this will select the closest unit/structure. 

Feel free to configure the radius in the Custom tab of the options. You can also configure if it should select buildings or not.
