# Battle Log

**Description:** Displays important battle events in a log

Using this handy list of events you will have an easy to way to jump to important locations on the map.
"Grunt is being attacked!"  .. Okay thanks, let me check my ONE THOUSAND grunts to see which one is getting damaged.
"T2 Unit detected" .. Good that you mention it, now let me go check each of these 3000 little icons on the minimap to see where it is.

NO MORE!

Features:
 
Easy to click list of important events. Easily find the spot a unit was destroyed or the spot an enemy unit was seen.
Important events such as "Commander destroyed" and "Nuclear missile Ready"
The first click will select units (when relevant), click again (or just double-click) to focus the camera on the unit.
Drag the "Battle Log" title to move the log to where you want it on the screen.
Also useful during spectating to easily jump from battle to battle

TECH FILTER:
At the top are 3 tech filter buttons, [T1] [T2] [T3] . Clicking these buttons will set the minimum tech level for the event to be logged, use them to once the T1 events become too much to follow / spammy.

SPACEBAR TO THE ACTION <-- use this, trust me!:
Press [Spacebar] without having any unit selected to instantly go to the location (or select the relevant unit) of the event. This is extremely powerful, once you hear the "Unit is being attacked" sound for example, you can press spacebar and instantly start to micro the unit.
