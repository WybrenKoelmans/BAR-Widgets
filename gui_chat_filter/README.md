# Chat Filter

**Description:** Censors, modifies or blocks chat messages based on user-defined rules.

You can adjust filter.txt to add words you want to filter.

Please note This filter only filters incoming messages on your PC,  outgoing chat is not modified!

Use  "Options -> Custom -> Chat Filter Mode" to have different options.
Censor turns bad words into ******
Block will stop the complete message from showing up
Kitty will replace bad words  with :X

Note that this previously included an optional AI enabled mode, which no longer functions. Local filtering works fine still.
