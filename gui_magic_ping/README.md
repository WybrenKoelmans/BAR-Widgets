# Magic Ping

**Description:** Invoking a default Ping (map marker) near a unit or thing, automatically labels it.

LOOK HERE! Using this widget if you do create a ping (map marker) close to something interesting, it will automatically label it. 

Writing anything will always take priority. This also works for pings from your allies (even if they don't have the widget) and spectators.
