# Lazy Reclaimers

**Description:** Reclaimers will stop reclaiming when the metal/energy storage is full.

It's very boring to micro reclaiming, but it makes a lot of sense to do it.

This widget will put reclaiming units on pause (if unselected) when the storage is full for the resource they are reclaiming. When storage is available again, it start reclaiming again.
