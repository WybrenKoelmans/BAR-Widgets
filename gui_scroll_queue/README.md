# Scroll Queue

**Description:** Use the mouse wheel to add/remove units from the factory build queue while hovering the build menu.

1. Hover your mouse over the unit you want to queue
2. (optionally hold any modifier like ALT, CTRL, etc)
3. Scroll your mouse wheel up for adding units to the queue, scroll down to subtract units.
