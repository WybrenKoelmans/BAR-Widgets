# Extended ETA Info

**Description:** Shows remaining metal cost below the build ETA

- **Author:** uBdead
- **Date:** 2026-04-18
- **License:** GNU GPL, v2 or later

## Details

When a unit is being built and the BuildETA widget is showing the estimated time of completion, this widget displays the remaining metal cost on the line below. This helps you quickly see how much metal is still needed to finish a structure or unit.
